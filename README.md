# Projects by Khushi Sharma

This repository contains two Python projects demonstrating Object-Oriented Programming (OOP) concepts and practical API usage for currency conversion.

---

## 1. OOPS_PY

A Python project focusing on **Object-Oriented Programming concepts** such as:

- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction

**Highlights:**

- Hands-on implementation of OOP principles.
- Includes multiple example classes and interactions.
- Useful for beginners and intermediates to strengthen OOP understanding.

---

## 2. Currency Converter

A Python-based **Currency Converter** project using a public API to convert amounts between currencies.

**Highlights:**

- Converts currencies dynamically using API data.
- Supports multiple currency pairs (e.g., EUR → GBP, USD → INR).
- Easy-to-use interface for quick conversions.
- Demonstrates real-world application of APIs in Python.
